Python Lists

A List is a built-in Python data structure used to store an ordered collection of items. Lists are mutable, meaning you can modify them (add, remove, or change items) after creation. They are defined using square brackets [] and can hold mixed data types (integers, strings, floats, etc.) and allow duplicate members.

Key Concepts

  • [] Defined using square brackets. Can hold mixed data types.
  • Mutable You can add, remove, or change items after creation.
  • 0-Indexed The first item is at index 0.
methods.py Python 3.10

my_list = ["A", "B"]

# Add to end

my_list.append("C")

# Remove last

my_list.pop()

# Change value

my_list[0] = "Z"

Python Tuples

A Tuple is a built-in Python data structure that is similar to a list but is immutable. This means once a tuple is created, its elements cannot be added, removed, or changed. They are defined using parentheses () and are often used for fixed collections of items, providing data integrity and slightly better performance than lists.

Feature List Tuple
Syntax [1, 2, 3] (1, 2, 3)
Mutable? Yes (Changeable) No (Fixed)
Performance Slower Faster
Methods Many (append, pop...) Few (count, index)
