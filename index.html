Python Lists

A List is a built-in Python data structure used to store an ordered collection of items. Lists are mutable, meaning you can modify them.

Key Concepts

  • [] Defined using square brackets.
  • Mutable Can change items after creation.
  • 0-Indexed First item is at index 0.
methods.py Python 3.10

my_list = ["A", "B"]

# Add to end

my_list.append("C")

# Remove last

my_list.pop()
